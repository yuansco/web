TinyEmbedded-Dual

Hardware
Firmware

TinyEmbedded-Dual is a compact Linux development board compatible with RISC-V D1s and ARM Cortex-A7 T113-S3 SoC. It features a convenient integrated USB-to-UART serial console.

TinyEmbedded

Hardware
Firmware

The TinyEmbedded is a low-cost embedded Linux development board that features an ARM9-based F1C200s microprocessor with an integrated USB-to-UART serial console.

Motion Angle

Software
Firmware

Motion Angle is a user-space program written in C, which uses the Linux IIO subsystem to read the three-axis acceleration of the g-sensor and use it to calculate the tilt angle of the board

USB Type-C Power Meter

Hardware
Firmware

The USB Type-C Power Monitor is a device that can monitor the voltage, current, and power of a Type-C device. It is composed of an STM32 MCU, A OLED display module and an INA3221 monitors IC.

eMMC to MicroSD Adapter

Hardware

This eMMC to MicroSD adapter is a small, compact device that allows you to connect an eMMC chip to your computer or other device that has a MicroSD slot, sach as Embedded Linux Board.

STM32G070 Evaluation Board

Hardware
Firmware

The STM32G070 Evaluation Board features an ARM Cortex-M0+ based STM32G070CBT6 MCU with an integrated USB to UART serial console and OLED module. It also support Zephyr OS.

STM32L431 Evaluation Board

Hardware
Firmware

The STM32L431 Evaluation Board features an ARM Cortex-M4 based STM32L431RCT6 MCU with an integrated USB to UART serial console and OLED module.

ComPort Tool

Software

ComPort-Tool is designed with C# and .NET Framework, and supports automatic time stamp, log saving, ASCII, hexadecimal, and binary display mode, and other convenient functions.

CheckSum Tool

Software

CheckSum Tool is a C# and .NET Framework application that calculates the 32-bit checksum of an input file. It also supports productivity functions such as drag-and-drop and copy button.

STM32F030 Evaluation Board

Hardware
Firmware

This STM32F030 evaluation board has a built-in USB to UART IC PL2303SA, which allows you to download code without any additional components. It also support Zephyr OS.
